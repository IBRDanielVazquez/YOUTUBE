# Videos — proyectos de producción

Cada documental vive en su propia carpeta aquí, nombrada con un slug corto: `videos/<slug>/`.

## Cómo empezar un documental nuevo

1. Crea `videos/<slug>/`.
2. Copia dentro los 10 archivos de `templates/` (mantén los mismos nombres, para que el orden de fases sea obvio).
3. Complétalos en orden, `00` a `09` — cada template indica al final cuál sigue.
4. Antes de dar cualquier fase por cerrada, revisa `research/` para el tema correspondiente; no avances con datos no verificados.
5. Antes de publicar, corre el checklist final de `CLAUDE.md`.

Ejemplo:

```
videos/
  el-canon-del-nuevo-testamento/
    00-brief-tema.md
    01-deep-research.md
    02-analisis-youtube.md
    03-angulo-guion.md
    04-guion-cinematografico.md
    05-storyboard.md
    06-produccion-higgsfield.md
    07-paquete-youtube.md
    08-shorts.md
    09-redes-sociales.md
```
