# YouTube Project — Biblical Documentary OS

Estudio de producción para el canal de YouTube de documentales bíblicos de nivel académico: investigación basada en evidencia, guion, storyboard, producción con Higgsfield MCP y paquete de distribución (YouTube, Shorts y redes sociales).

Las instrucciones completas de operación están en [`CLAUDE.md`](./CLAUDE.md) — léelo antes de trabajar en este repositorio.

## Estructura

- **`research/`** — espejo de la biblioteca NotebookLM (16 categorías), fuente principal de toda investigación.
- **`templates/`** — un template en blanco por cada una de las 11 fases de la metodología.
- **`videos/`** — un folder por documental, instanciado a partir de `templates/`. Ver `videos/README.md` para el flujo de trabajo.

## Configuración

Para comenzar a trabajar en este proyecto, asegúrate de establecer este directorio como tu espacio de trabajo activo en tu editor de código.
