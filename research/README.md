# Research — espejo de la biblioteca NotebookLM

Esta carpeta refleja, categoría por categoría, la biblioteca NotebookLM que es la fuente principal de todo el proyecto (ver `CLAUDE.md`). Antes de investigar cualquier tema para un documental, revisa aquí primero.

Cada subcarpeta corresponde a un notebook de NotebookLM. Guarda ahí:

- Notas y resúmenes extraídos del notebook.
- Citas y referencias exactas (autor, obra, página/línea) para poder verificar cualquier afirmación.
- Enlaces o identificadores de las fuentes originales (papers, libros, bases de datos, excavaciones).
- Clasificación de cada afirmación relevante como Hecho / Hipótesis / Interpretación / Tradición / Consenso académico.

No se sustituye la investigación en NotebookLM — esta carpeta es el puente entre esa biblioteca y el trabajo de guion en `videos/`.
